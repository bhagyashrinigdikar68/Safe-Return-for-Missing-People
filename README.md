# Safe Return for Missing People

# Intelligent System for Finding Missing People using Face

This project is an AI-powered system designed to help identify and locate missing individuals using facial recognition and eye-state detection from images, CCTV footage, or real-time camera inputs.

An AI-based system that helps identify missing people using **face detection**.

## Features
- Face detection
- Matching with missing person database
- Notification system

## Tech Stack
- Python
- OpenCV
- DeepFace
- Flask
- ReactJS
- Maven (Spring Boot for notifications)

## Installation

Clone the repository:

git clone https://github.com/bhagyashrinigdikar68/Safe-Return-for-Missing-People.git

cd Safe-Return-for-Missing-People

Install requirements:

pip install -r requirements.txt

## Run Python Services

python api.py

python app.py

python app1.py

python app3.py

## Run Notification Service

mvn.cmd spring-boot:run -f safe-return-notifications/pom.xml
